# Crylic

Crylic은 마인크래프트 커스텀 런처입니다.  
기본적인 기능부터 시작하여 점차 많은 기능을 지원할 예정입니다.

## 개발

### 개발 환경 세팅

[Node.js](https://nodejs.org) v24.11에서 구동 및 빌드가 가능하며 에디터로 [Visual Studio Code](https://code.visualstudio.com/)를 권장합니다.  
패키지 매니저는 Yarn을 사용합니다.

### Visual Studio Code 필수 확장 프로그램

- [Vue](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

### 테스트 구동

아래의 명령어로 필요한 라이브러리를 설치합니다.

```
yarn
```

라이브러리 설치가 완료되면 아래의 명령어로 테스트 환경에서 구동합니다.

```
yarn start
```
